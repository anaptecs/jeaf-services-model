# JEAF Services API Model README #

This repository contains the UML model for JEAF Services API project. Please be aware that the actual model is under version control of MagicDraw Teamwork Server.

## Links ##
For further information please refer to:

* [JEAF Services](https://anaptecs.atlassian.net/l/c/wuJ871mE)
* [Maven Repository](https://search.maven.org/artifact/com.anaptecs.jeaf.uml/jeaf-services-api-model)


## How do I get set up? ##

* Create a clone of this repository on your local machine.
* Execute Maven on the top level project `maven clean install`
